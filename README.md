[![Typing SVG](https://readme-typing-svg.herokuapp.com?font=Fira+Code\&size=24\&duration=8000\&pause=1000\&color=5E81AC\&width=435\&lines=INSTALLATION)](https://git.io/typing-svg)


1. Clone this repository:

```bash
git clone https://github.com/shreyas-sha3/niri-dots.git ~/.niri-dots
cd ~/.niri-dots
```

2. Make the installer executable:

```bash
chmod +x install.sh
```

3. Run the installer:

```bash
./install.sh
```

* It will **check and install missing packages** via `paru`.
* Sync all dotfiles.
* Install and apply **Nord GTK theme** and **Nordzy icons**.
* Apply wallpaper automatically.
* **For Zen Theme:** install and apply **Zen userChrome.css** and enable custom CSS: `about:config` → `toolkit.legacyUserProfileCustomizations.stylesheets` → `true`.

4. Restart your session or apps if needed to see theme changes.
